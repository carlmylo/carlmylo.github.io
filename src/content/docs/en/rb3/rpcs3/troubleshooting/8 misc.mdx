<details>
<summary>Scrolling through the library has long pauses.</summary>
You didn’t read the guide and did not set the “Network Status” to “Connected” in the [[Network tab when setting up the Custom Configuration]](https://rb3pc.milohax.org/custom_config_net) for Rock Band 3.
</details>

<details>
<summary>My game doesn't fill the screen.</summary>
Activate "**Overscan**" in Rock Band 3.  
To do this, go to:  
`Menu > Options > System Settings > Overscan`
</details>

<details>
<summary>I can't drag and drop PKGs into RPCS3.</summary>
This is a known issue with RPCS3. Restart it and it will should work.
</details>
